# 🌦 Weather Dashboard App

A modern weather dashboard built using Python and CustomTkinter.

## 🚀 Features
- 🌡 Real-time weather data
- 📅 5-day forecast
- 📍 Auto location detection
- 🌙 Dark/Light mode
- 🎨 Modern UI with cards layout

## 🛠 Tech Stack
- Python
- CustomTkinter
- OpenWeatherMap API

## ▶️ How to Run
1. Install dependencies:
pip install customtkinter requests geocoder pillow

2. Add your API key in the code

3. Run:
python weather_dashboard.py
